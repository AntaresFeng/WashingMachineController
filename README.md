# WashingMachineController
**Washing machine controller based on msp430.**

### 洗衣机控制器设计

#### 所有题目要求：
1. 开机显示“2023年MSP430单片机课程设计”；
2. 开机显示两位同学的学号，专业，姓名（汉字点阵显示），完成日期；
3. 再通过按键切换显示题目要求（汉字点阵显示）；
4. 自主想出至少一项发挥要求。

#### 基本要求：

1. 基本功能包括：自动洗衣流程：洗涤->漂洗->脱水->漂洗->脱水->关机；
2. 用LCD显示当前状态/状态剩余洗涤时间；
3. 用键盘设计启动、暂停、停止键，在洗衣过程中键都有效；
4. 设计一个“预约”键，用于定时启动洗衣机；
5. 洗涤完成后用声音提示。

#### 发挥部分：

1. 研究不同模式下的洗涤过程（棉普通方式、羊毛、浸泡等功能）；
2. 设计一个水位键，用于指定不同水位。
